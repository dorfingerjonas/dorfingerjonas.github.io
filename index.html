Work

Tic Tac Toe

Tic Tac Toe is a well-known game, I decided to create my own game, three gamemodes to make the game more attractive. You are able to play the game alone against a bot (gamemode in progress), you can play online with your friend in realtime. It's also possible to play TicTacToe on one device with two people. This game is only a fun project in my leisure time, I wanted to use this project to learn more about online game development an database-client communication. Website

Schuldners Liste

Schuldners Liste is a mobile application made with web technologies. The app is used to help you to keep in mind which persons have debts to you. You have the opportunity to restore deleted entries, this is useful for example if you have deleted the wrong entry. The user of the app can choose his theme. There are several predefined themes from which the user can select his current theme. It is planned that it should be possible to have the themes created by users individually. Website
